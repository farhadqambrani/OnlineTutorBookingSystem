# Learn Hub – Online Tutor System

## 📌 Project Overview
Learn Hub is a web-based Online Tutor System developed using ASP.NET, C#, SQL Server, and Visual Studio. The system helps students connect with tutors through an easy-to-use online platform.


## 🚀 Features
- Student Registration
- Tutor Registration
- Secure Login & Authentication
- Course Management
- Student Dashboard
- Tutor Dashboard
- Admin Panel
- Responsive User Interface


## 🛠 Technologies Used
- ASP.NET
- C#
- SQL Server
- HTML
- CSS
- JavaScript
- Visual Studio


## 🎯 Project Objective
The main objective of Learn Hub is to provide a simple and efficient online learning environment for students and tutors.
